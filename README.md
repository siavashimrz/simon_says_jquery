# Simon Says Game

A simple Simon Says game built with HTML, CSS, JavaScript, and jQuery as part of a practice project from App Brewery.

## Play It Here!

https://siavashimrz.github.io/simon_says_jquery/

## Features

* Random sequence generation
* Increasing difficulty as levels progress
* Visual button animations
* Sound effects for each color
* Game over animation and sound
* Restart the game by pressing any key

## Technologies Used

* HTML
* CSS
* JavaScript
* jQuery

## How to Play

1. Press any key to start the game.
2. Watch the sequence played by Simon.
3. Repeat the sequence by clicking the colored buttons.
4. Complete the sequence correctly to advance to the next level.
5. Make a mistake and the game ends.

## How to Run

Clone the repository and open `index.html` in your browser.

No build tools or dependencies are required.

## About This Project

This project was created for learning and practice purposes.

The goal was to practice:

* DOM manipulation
* Event handling
* jQuery
* Animations
* Game logic

The project is intentionally simple and is not intended to be a production application.
